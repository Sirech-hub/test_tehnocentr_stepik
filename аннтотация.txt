Для реализации автоматизированного тестирования https://tehnocentr.ru/ 
(специализированный сайт запчастей и комплектующих для электроники)
используются Python, Selenium WebDriver и фреймворк pytest. 
Структура тестов организована по паттерну PageObject
Используются проверки по значению обьектов и создание скриншотов на этапах.
Реализованы основные пути прохождения клиента от регистрации до выбора товара (возможно рандомно добавить любой товар) и пособа доставки и оплаты товара
в проекте использовалась реальная учетная запись поэтому некоторые пути не реализованы из-за конфиденциальности данных.


python -m pytest -v -s test_login.py
python -m pytest -v -s test_basket_product.py
python -m pytest -v -s test_catalog_product.py
